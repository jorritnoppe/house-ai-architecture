# Telemetry Data Flow

## Role of telemetry
Historical and live measurements are used to answer questions, compare behavior over time, and support future anomaly detection.

## Typical flow
data source
-> ingestion
-> time-series storage
-> query helpers
-> summaries and comparisons
-> AI-facing answer generation

## Typical use cases
- room climate summaries
- power usage comparisons
- automation verification
- device health observations
